# Glossary

## Account

A banking relationship or product, identified by an account ID, that can expose account details, balances, and transactions through the API.

## Balance

The amount of money reported for an account at a point in time, including currency and balance type information where applicable.

## Transaction

A posted or pending movement of money on an account, such as a card purchase, transfer, fee, deposit, or payment.

## Consent

A customer authorization record that describes which account-information or payment capabilities an API client is allowed to use and when that authorization expires or is revoked.

## Payment instruction

A request to initiate a payment from a debtor account to a creditor account, including amount, currency, creditor details, remittance information, and status.

## OpenAPI

A machine-readable API description format used to document HTTP endpoints, schemas, parameters, responses, and authentication requirements.
